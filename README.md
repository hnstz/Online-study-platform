# Online-study-platform
Проект представляет собой веб-приложение для просмотра и фильтрации онлайн-курсов.

**Ссылка на деплой:** https://online-platform-gamma.vercel.app

## Технологический стек
1) HTML5: разметка структуры приложения.

2) SCSS: препроцессор для стилизации с использованием переменных и вложенности.

3) JavaScript (ES6+): реализация логики фильтрации, поиска и динамического рендеринга контента.

4) Vercel: среда деплоя проекта.

5) Инструменты разработки:

  а) ESLint: статический анализ кода для выявления ошибок и соблюдения стандартов.
  
  б) Prettier: автоматическое форматирование кода для поддержания единого стиля.

## Основная функциональность
Динамический рендеринг: вывод карточек курсов на основе данных из data.js.

Фильтрация: фильтрация списка курсов по категориям.

Поиск: поиск курсов по названию.

Комбинированный поиск: совместная работа фильтрации по категории и текстового поиска. Поиск осуществляется только среди отфильтрованных по категории курсов.

## Реализованные исправления и обновления
1) Исправлена опечатка в именовании "product".
   
2) Количество отображаемых категорий приведено в соответствие с фактическими данными.

3) Реализована связная логика работы фильтров и поиска: при активном фильтре категории поиск выполняется только в рамках выбранной категории.

## Локальный запуск

### 1. Подготовка
Убедитесь, что у Вас установлены Node.js и npm (они нужны для работы с зависимостями проекта). Проверить их наличие можно в терминале командами:
```bash
node -v
npm -v
```

### 2. Клонирование и установка
Откройте терминал в нужной папке и клонируйте репозиторий:
```bash
git clone https://github.com/hnstz/Online-study-platform
```
Перейдите в папку проекта:
```bash
cd Online-study-platform
```
Установи зависимости (хотя проект на чистом JS, они необходимы для работы линтера, если он есть):
npm install

### 3. Запуск

Использование Live Server (рекомендуется для разработки). 

Если установлен Visual Studio Code и расширение Live Server. Нажмите кнопку "Go Live" в правом нижнем углу — проект откроется в браузере, и страница будет обновляться автоматически при каждом сохранении файлов.
